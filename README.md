# LoveSpace 💖

**LoveSpace** — это интерактивное пространство для пар, созданное в формате Telegram Mini App. Приложение помогает укреплять отношения через геймификацию, совместное планирование и обмен эмоциями.

## ✨ Основные функции

- **💎 Система заданий:** Поручайте друг другу приятные дела, выполняйте их и зарабатывайте кристаллы.
- **🎁 Магазин желаний:** Тратьте заработанные кристаллы на желания, которые подготовил для вас партнер.
- **🎡 Колесо фортуны:** Настраиваемое колесо для принятия совместных решений (куда пойти, что посмотреть, кто готовит ужин).
- **📖 Лента воспоминаний:** Общий таймлайн ваших памятных моментов с фото и эмодзи.
- **💭 Мониторинг настроения:** Делитесь своим состоянием и статусом на день. Партнер всегда будет знать, как вы себя чувствуете.
- **❓ Вопрос дня:** Ежедневные вопросы, которые помогают лучше узнать друг друга и начать интересный диалог.
- **🗓 Планировщик:** Совместные планы на день и важные даты.
- **🔔 Telegram-уведомления:** Получайте мгновенные оповещения о действиях партнера и утренние напоминания о планах через бота.

## 🎨 Дизайн и темы

Приложение поддерживает 3 современных темы:
1. **Light** — классический светлый интерфейс.
2. **Dark** — комфортный темный режим.
3. **Glass** — премиальный "стеклянный" эффект (Glassmorphism) с размытием фона.

Также доступен выбор акцентных цветов: Розовый, Голубой, Зеленый, Оранжевый, Фиолетовый и Красный.

## 🛠 Технологический стек

- **Frontend:** Single-file HTML5 / CSS3 (Vanilla) / JavaScript.
- **Backend:** Node.js Serverless Functions (Vercel).
- **Database:** Upstash Redis (для высокой скорости и масштабируемости).
- **Bot:** Telegram Bot API для уведомлений и Cron-задач.

## 🚀 Развертывание

Приложение оптимизировано для деплоя на **Vercel**.

1. Клонируйте репозиторий.
2. Настройте переменные окружения в Vercel:
   - `UPSTASH_REDIS_REST_URL`
   - `UPSTASH_REDIS_REST_TOKEN`
   - `TELEGRAM_BOT_TOKEN`
3. Настройте Webhook для вашего бота на URL вашего приложения.
